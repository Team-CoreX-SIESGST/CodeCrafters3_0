Flow Guardian Desktop App

How to run:
1. Open the folder named "FlowGuardian".
2. Double-click "FlowGuardian.exe".
3. Allow camera access if you want eye and face tracking.

Notes:
- Keep the "_internal" folder beside the exe. Do not move the exe out by itself.
- This is a packaged Windows desktop build of the os-observer-backend project.
- The app launches the same monitoring overlay and local API used by the backend.
